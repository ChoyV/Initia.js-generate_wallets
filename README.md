This package is to generate wallets in Initia network

/// USER GUIDE \\\

1) Open the terminal
2) npm i
3) npx ts-node generate.ts {NUMBER OF WALLETS YOU NEED} (for ex. "npx ts-node generate.ts 5")
4) Check your generated wallets in keys.json
